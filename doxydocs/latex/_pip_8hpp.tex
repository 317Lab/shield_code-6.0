\doxysubsection{Detailed Description}
Header file for the \doxylink{class_pip}{Pip} library for Dartmouth\textquotesingle{}s 317 Lab. 

This library manages the voltage sweep on the PIP sensors using the Arduino Due\textquotesingle{}s onboard DAC. It is basically a vastly simplified version of the \doxylink{class_pip}{Pip}, DAC, and Max1147 libraries in the Isinglass/\+Apophis shield code. Written by Max Roberts in 2015. We are able to use this simplified version because of how nice the Due is \+:)
