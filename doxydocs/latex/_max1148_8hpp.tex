\doxysubsection{Detailed Description}
Header file for the \doxylink{class_max1148}{Max1148} library for Dartmouth\textquotesingle{}s 317 Lab. Manages ADC communication through SPI. 

Basically just the Max1147 library from the Isinglass/\+Apophis shield code rewritten for the Arduino Due. Written by Jacob at some point unknown to me.

See the data sheet for more information -\/ \href{https://www.analog.com/media/en/technical-documentation/data-sheets/MAX1146-MAX1149.pdf}{\texttt{ https\+://www.\+analog.\+com/media/en/technical-\/documentation/data-\/sheets/\+MAX1146-\/\+MAX1149.\+pdf}}
